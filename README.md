# 🌟 My Portfolio

A modern, responsive personal portfolio website built with **React.js**, following the latest design trends to showcase skills, projects, and experience.

---

## 📖 About

This portfolio is designed to provide a clean and engaging overview of my work as a developer. It features a contemporary UI/UX with smooth interactions and a mobile-friendly layout.

---

## 🛠️ Built With

- **React.js** – Component-based UI library for building interactive interfaces
- **CSS3 / Modern Styling** – Responsive, trend-following visual design
- **JavaScript (ES6+)** – Core application logic

---

## ✨ Features

- ⚡ Fast and responsive single-page application
- 🎨 Trending, modern UI design
- 📱 Mobile-friendly layout
- 🔗 Sections for About, Skills, Projects, and Contact

---

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v14 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

```bash
# Clone the repository
git clone https://github.com/ngocbinh123/my-portfolio.git

# Navigate into the project directory
cd my-portfolio

# Install dependencies
npm install

# Start the development server
npm start
```

The app will run at `http://localhost:3000`.

### Build for Production

```bash
npm run build
```
